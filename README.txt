The interbank network visualiser allows for network visualisation based on user provided input data. Specifics are given in the user guide.

Quick start: 
- Run the programme with 'python app.py' from the command line
- Open the local server as shown in the command window (e.g. http://127.0.0.1:5000/) in Chrome or Firefox
- Upload csv files from the test_data file (Matrix_201609_DE, Matrix_201701_DE, Testdatensatz_Bankdaten_201609_DE, Testdatensatz_Bankdaten_201701_DE)
